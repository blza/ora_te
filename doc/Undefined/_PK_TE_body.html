Package PK_TE

The package that provides functions to substitute values instead of placeholders.
Only functions and types that are to be called by end user are exposed. Some supporting functions are moved to PK_TE_IMPL.

SUBSTITUTE

 CLOB SUBSTITUTE(A_TE IN TY_TE, 
                 A_CURSOR IN PK_TE.REFCUR, 
                 A_CONCAT_BY IN VARCHAR2 DEFAULT NULL)
Replaces placeholders in compiled Template Expression with values from cursor in iteration.
Each individual substitution is concatenated with the following with optional a_concat_by in between.
Cursor must return instances of ty_p or ty_m UDTs otherwise the function will throw.
Parameters:
A_TE - ty_te compiled template expression
A_CURSOR - a cursor that must return either instances of ty_m or instances of ty_p
A_CONCAT_BY - if present each individual substitutions are concatenated in a loop with a_concat_by in between
Returns:
clob - a large character lob with substituted values (if any)
Throws:
EX_TE_IS_NULL - if null template expression is passed
EX_CURSOR_OF_WRONG_TYPE - if passed cursor does not return instances of ty_m or instances of ty_p
