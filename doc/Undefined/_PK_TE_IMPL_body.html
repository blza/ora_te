Package PK_TE_IMPL

Supporting functions that help to implement PK_TE functionality (mostly dealing with recursive calls to handle loop structures). Moved to separate package to expose in PK_TE specification only those types and functions that are to be called by end user.

Replaces placeholders in compiled Template Expression with values from cursor in iteration.
Caches results returned from cursor so there's no need to reopen it if more than one loop structures refering this cursor are defined in template expression.
Each individual substitution is concatenated with the following with optional a_concat_by in between.
Cursor must return instances of ty_p or ty_m UDTs (depending on ty_te type) otherwise the function will throw.
Parameters:
A_LOOP_IDX - loop index (1-9) to select appropriate cache
A_TE - ty_te compiled template expression
A_CURSOR - a cursor that must return instances of ty_p or of ty_m
A_CONCAT_BY - if present each individual substitutions are concatenated in a loop with a_concat_by in between
A_C1 - a sys_refcursor to be used in loop structure substitution
A_C2 - a sys_refcursor to be used in loop structure substitution
A_C3 - a sys_refcursor to be used in loop structure substitution
A_C4 - a sys_refcursor to be used in loop structure substitution
A_C5 - a sys_refcursor to be used in loop structure substitution
A_C6 - a sys_refcursor to be used in loop structure substitution
A_C7 - a sys_refcursor to be used in loop structure substitution
A_C8 - a sys_refcursor to be used in loop structure substitution
A_C9 - a sys_refcursor to be used in loop structure substitution
A_P_CACHE - a cache that stores results of cursor fetches returning ty_p instances
A_M_CACHE - a cache that stores results of cursor fetches returning ty_m instances
Returns:
clob - a large character lob with substituted values (if any)
Throws:
EX_TE_IS_NULL - if null template expression is passed
EX_CURSOR_OF_WRONG_TYPE - if passed cursor does not return instances of ty_p
CEX_TE_OF_WRONG_TYPE - if passed compiled template expression is not of type ty_te.EL_NUMBERED()
