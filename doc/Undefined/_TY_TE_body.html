Object Type TY_TE

TY_TE (Template Expression)
A type that represents a compiled template expression.
By template expression (TE) we understand a string with numbered or named placeholders.
By compiling TE we understand parsing it, finding numbered or named placeholders and storing the whole expression in internal
structures (a nested table of ty_sph).

                                        A_LOOP_PH_BEGIN IN VARCHAR2 DEFAULT '{%', 
                                        A_LOOP_PH_BODY IN VARCHAR2 DEFAULT '%', 
                                        A_LOOP_PH_END IN VARCHAR2 DEFAULT '%}')
Parse string for named or numbered templates
Stores result of parsing in compiled_template_ nested table.
Parameters:
A_TEMPLATE_STRING - a template string
A_PH_START - a string that denotes the beginning of placeholder
A_PH_END - a string that denotes the end of placeholder
A_TYPE - a type of placeholders to search in string. Accepts either ty_te.EL_NUMBERED() or ty_te.EL_NAMED()
A_LOOP_PH_BEGIN - a string that denotes the beginning of loop structure within template expression
A_LOOP_PH_BODY - a string that separates loop inner tempate expression from cursor number and from loop structure options
A_LOOP_PH_END - a string that denotes the end of loop structure within template expression
Returns:
The instance of template expression (ty_te) or null if a_type != ty_te.EL_NUMBERED() and a_type != ty_te.EL_NAMED()
or if no teplate expressions were found in string.
Also returns null if there's misuse of placeholder denotation strings (a_ph_start, a_ph_end, a_loop_ph_begin, a_loop_ph_body, a_loop_ph_end).
